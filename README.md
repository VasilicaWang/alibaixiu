### 项目架构 
- 系统分层: 
    - 数据层 mongoDB
    - 服务层：node.js(express)
    - 客户端：art-template、jQuery、font、swiper
- 实现功能：后台
    - 登录功能
        - 登录注销功能
        - 登录拦截
    - 用户操作功能
        - 添加用户、用户头像上传
        - 用户列表展示
        - 用户信息修改
        - 删除用户
        - 批量删除用户
        - 修改密码
    - 文章分类
        - 添加文章分类
        - 修改
        - 删除文章分类
    - 文章操作功能
        - 上传文章封面图
        - 添加文章
        - 文章列表数据渲染
        - 分页功能
        - 筛选功能
        - 编辑文章
        - 删除文章
    - 评论
        - 评论列表展示
        - 评论列表分页功能
        - 评论审核功能
    - 数量统计
    - 图片轮播
        - 数据添加
        - 删除
    - 网站设置
    - 更改当前用户头像
- 前台实现功能：
    - 轮播图数据展示
    - 热门推荐部分
    - 最新发布文章
    - 随机推荐
    - 最新评论
