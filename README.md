### 项目架构 
- 系统分层: 
    - 数据层 mongoDB
    - 服务层：node.js(express)
    - 客户端：art-template、jQuery、font、swiper
- 实现功能：
    - 登录功能
        - 登录注销功能
        - 登录拦截
    - 用户操作功能
        - 添加用户、用户头像上传
        - 用户列表展示
        - 用户信息修改
        - 删除用户
        - 批量删除用户
        - 修改密码
    - 文章分类
        - 添加文章分类
        - 修改
        - 删除文章分类
    - 文章操作功能
        - 上传文章封面图
        - 添加文章
        - 文章列表数据渲染
        - 分页功能
        - 筛选功能
        - 编辑文章
